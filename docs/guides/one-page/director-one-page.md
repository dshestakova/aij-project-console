# AIJ Project Console: краткий гайд для директоров

Как быстро смотреть портфель, аналитику и нагрузку по CSM

## Для чего использовать
- Смотреть состояние портфеля активных AIJ-проектов.
- Видеть распределение проектов по отраслевым управлениям.
- Проверять нагрузку и структуру портфеля по CSM.
- Контролировать флагманы и готовность паспортного контура.
- Переходить из аналитики в отфильтрованный реестр проектов.

## Главные разделы
- **Дашборд** - общая картина по активным проектам.
- **Проекты** - поиск, фильтры, карточки и выгрузка реестра.
- **Аналитика** - статусы, ОУ, CSM-матрица, директорская структура и флагманы.
- **AI-аналитик** - пока не подключен и будет доступен позже.

## Что смотреть в аналитике
- **Статусы** - где сконцентрированы проекты по стадиям.
- **Отраслевые управления** - как распределен портфель.
- **CSM-матрица** - проекты по CSM; цвет показывает статус, обводка - флагманский контур.
- **Директорская структура** - проекты по ОУ и CSM, средняя нагрузка.
- **Флагманы** - описание, паспорт, инновационность, ПРБР и одобрение ЦА.

## Быстрые сценарии
- **Свое ОУ:** Аналитика -> фильтр ОУ -> CSM-матрица или директорская структура.
- **Конкретный CSM:** Проекты -> фильтр CSM -> открыть нужные карточки.
- **Флагманы:** Проекты -> «Флагман: да» или Аналитика -> флагманская аналитика.
- **Отчет:** «Скачать» у нужного блока аналитики или «Скачать реестр» в проектах.

## Как читать обозначения
- Цвет плашки проекта = текущий статус.
- Черная обводка = флагман или паспортный контур.
- Архивные проекты не входят в обычную аналитику.
- «Описание загружено» = CSM подготовил описание флагманского проекта.

## Если что-то не видно
- Сбросьте или проверьте активные фильтры.
- Проверьте, не включен ли режим архива.
- Если раздел или данные недоступны, обратитесь к администратору.
